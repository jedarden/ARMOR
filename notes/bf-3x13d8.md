## Workflow Architecture

**Argo WorkflowTemplate: `armor-drift-check`**
```yaml
schedule: 0 9 * * *  # Daily at 9 AM UTC
entrypoint: drift-check
steps:
  - run-check: Clone ARMOR repo, run drift check script
  - archive-results: Log results to Argo
```

**Parameters:**
- `git-repo`: jedarden/ARMOR
- `branch`: main
- `releases-threshold`: 50
- `days-threshold`: 30
- `slack-webhook-url`: (configured via Secret or parameter)
- `dry-run`: false

**Resources:**
- Request: 500m CPU, 512Mi memory
- Limit: 2000m CPU, 2Gi memory
- Timeout: 10 minutes
- Retry: OnError, 2 attempts with exponential backoff
