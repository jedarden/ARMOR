## Key Method: `_has_key_token`

The `_has_key_token(line: str) -> bool` method (lines 418-492) correctly identifies when a line contains a key token (a colon that starts a key-value pair).

### What It Does

1. **Distinguishes key-bearing `:` from scalar `:`**
   - Ignores colons inside single/double quotes
   - Ignores colons inside flow collections (`{}` and `[]`)
   - Ignores colons in comments (after `#`)
   - Ignores colons after block scalar indicators (`|` or `>`)

2. **Validates key context**
   - Checks that text before the colon is a valid key
   - Ensures colon is followed by whitespace or end of line
   - Handles sequence dash prefixes correctly

3. **Marks lines with key tokens**
   - Used by `_classify_line_type()` to classify lines as `KEY_BEARING`
   - Used by `record_indent_transition()` to track scope changes
   - Available for downstream classification via line classification
