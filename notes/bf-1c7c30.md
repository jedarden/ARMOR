2. **Multiple Response Types**: Supports both:
   - `*httptest.ResponseRecorder` (for unit tests)
   - `*http.Response` (for integration tests)

3. **Variant Functions**:
   - `ValidateContentTypeAny()` - Validate against multiple allowed content-types
   - `ValidateContentTypePrefix()` - Validate content-type starts with prefix
   - Convenience functions: `ValidateContentTypeJSON`, `ValidateContentTypeXML`, `ValidateContentTypeText`, etc.

4. **Non-asserting Versions**:
   - `CheckContentType()` - Returns boolean for conditional logic
   - `CheckContentTypeAny()` - Boolean version of multiple type check
   - `CheckContentTypePrefix()` - Boolean prefix check

5. **Content-Type Analysis Helpers**:
   - `GetContentTypeCharset()` - Extract charset parameter
   - `GetContentTypeWithoutParams()` - Get base MIME type without parameters
   - `IsContentTypeJSON()`, `IsContentTypeXML()` - Type checking helpers
