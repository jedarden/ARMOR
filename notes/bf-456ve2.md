## Key Features

### Error Injection Control

```go
// Frequency-based injection (30% of requests)
scenario := NewRateLimitScenario(0.3, 5*time.Second)

// Deterministic with seed
server := NewEnhancedErrorServerWithSeed(scenarios, 12345)

// Conditional injection
scenario.Condition = func(r *http.Request) bool {
    return r.URL.Path == "/trigger"
}
```

### Statistical Tracking

```go
total, errors, rate := server.GetStats()
// Returns: (100, 30, 0.3) for 30% error rate

server.ResetCounters() // Reset for new test phase
```

```go

```
