# Pluck Minimum Version Requirements - Documentation

## Executive Summary

**Pluck is not a standalone package or tool.** It is a **strand (component) within the NEEDLE project** (`github.com/jedarden/NEEDLE`). Therefore, "Pluck version requirements" are actually the **NEEDLE project requirements**.

## Source: NEEDLE Project

- **Repository**: `github.com/jedarden/NEEDLE`
- **Pluck Source**: `/home/coding/NEEDLE/src/strand/pluck.rs`
- **Purpose**: Pluck is the primary bead selection strand that processes >90% of all beads in NEEDLE

## Minimum Version Requirements

### 1. Rust Toolchain (MSRV)

| Requirement | Minimum Version | Source |
|-------------|-----------------|--------|
| **Rust** | **1.75** (2023-12-28) | `/home/coding/NEEDLE/rust-toolchain.toml` |
| **Rust Edition** | 2021 | `Cargo.toml` |
| **NEEDLE Version** | 0.2.11 | `Cargo.toml` |

### 2. Core Rust Dependencies (from NEEDLE Cargo.toml)

| Dependency | Minimum Version | Purpose |
|------------|-----------------|---------|
| `tokio` | 1 | Async runtime |
| `serde` | 1 | Serialization |
| `serde_json` | 1 | JSON support |
| `serde_yaml` | 0.9 | YAML support |
| `clap` | 4 | CLI parsing |

### 3. Pluck-Specific Configuration

**Config File**: `.needle.yaml` (workspace-level configuration)

**Relevant Settings**:
```yaml
strands:
  pluck:
    exclude_labels: []          # Labels to exclude from selection
    split_after_failures: 0      # Auto-split threshold (0 = disabled)
```

**Documentation Reference**: See `docs/pluck-debug-configuration.md` for debug logging options.

## Dependencies Without Clear Minimum Versions

The following dependencies from NEEDLE's indirect dependencies do **not have explicit minimum versions documented** in the source files:

| Dependency | Status |
|------------|--------|
| `aws-sdk-go-v2` crates (indirect) | Minimum version not documented in ARMOR workspace |
| `gopkg.in/yaml.v3` (indirect) | Transitive dependency; version controlled by Cargo.lock |

**Note**: These are indirect dependencies resolved by Cargo's dependency resolution. The Cargo.lock file pins exact versions for reproducible builds.

## Verification Method

To verify NEEDLE/Pluck installation compatibility:

```bash
# Check Rust version
rustc --version  # Should be >= 1.75

# Check NEEDLE installation
needle --version

# Verify Pluck strand is available
needle run --help | grep -i strand
```
